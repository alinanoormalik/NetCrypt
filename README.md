# NetShield

Welcome to NetShield! 🌐🛡️

A collection of networking and network security tools built with Python. This repository focuses on packet analysis, Wi-Fi monitoring, and basic network security concepts.
More tools and improvements will be added over time.

---
# Project Structure
## What's Inside?

### Guardify – Wi-Fi Device Monitor & Rogue AP Detector
A network monitoring tool that scans connected devices, identifies unauthorized devices, detects rogue routers, and stores device history using ARP scanning techniques.

### SnifferX – Packet Sniffer
A Python-based packet sniffer that captures TCP, UDP, and ICMP traffic and displays packet information including source/destination IPs and raw data.
